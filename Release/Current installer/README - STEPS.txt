Βήμα 1: Εκτελέστε τον installer (Setup Version 3.2 - Xanax.exe) και ολοκληρώστε την εγκατάσταση. 
Βήμα 2: Όταν η εγκατάσταση ολοκληρωθεί επιτυχώς εκτελέστε το setupDefaultDB.cmd ώστε να έχετε την ενδεικτική βάση δεδομένων που ετοιμάσαμε για την διευκόλυνση σας (Στην περίπτωση κανονικού χρήστη δεν απαιτείται αυτό το βήμα. Η εφαρμογή θα δημιουργήσει μια κενή βάση στην πρώτη εκτέλεση της)

Η εγκατάσταση και η ρύθμιση ολοκληρώθηκε.
Για απεγκατάσταση εκτελέστε τον uninstaller (Uninstaller.cmd) ως διαχειριστής. (Δεξί κλικ -> Εκτέλεση ως διαχειριστής)
